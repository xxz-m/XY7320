# 标签字典

本文档定义项目使用的所有 Obsidian 标签，方便搜索和 Dataview 查询。

## 主题标签

| 标签 | 说明 |
|---|---|
| `#stm32` | STM32 芯片相关 |
| `#bootloader` | Bootloader 开发 |
| `#app-fw` | APP 固件开发 |
| `#mota` | mOTA 组件 |
| `#uart` | 串口通信 |
| `#flash` | Flash 操作/分区 |
| `#cmake` | CMake 构建 |
| `#qt` | Qt 上位机 |
| `#upgrade` | 升级协议/流程 |
| `#linker` | 链接脚本 |
| `#gcc` | GCC 工具链 |
| `#dma` | DMA 传输 |
| `#freertos` | FreeRTOS 相关 |
| `#architecture` | 架构设计 |
| `#project` | 项目管理 |

## 类型标签

| 标签 | 说明 |
|---|---|
| `#type/ref` | 技术参考文档 |
| `#type/guide` | 操作指南 |
| `#type/log` | 开发日志 |
| `#type/plan` | 规划文档 |
| `#type/issue` | 问题排查 |
| `#type/checklist` | 验收清单 |
| `#type/moc` | Map of Content（索引） |

## 状态标签

| 标签 | 说明 |
|---|---|
| `#status/current` | 当前阶段 |
| `#status/completed` | 已完成 |
| `#status/planned` | 计划中 |

## 使用示例

在 Obsidian 搜索框中输入：

```text
tag:#upgrade
```

可快速找到所有升级相关文档。

```text
tag:#type/ref
```

可筛选所有技术参考文档。

```text
tag:#bootloader AND tag:#type/guide
```

可查找 Bootloader 操作指南。
