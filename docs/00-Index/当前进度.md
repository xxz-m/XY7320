# 当前进度

> [!success]
> 这里不记录所有历史，只记录当前主线进度和最近几个关键变化。

## 当前主线状态

- 固定码握手升级链路已跑通
- APP 新分层框架已落地第一版
- Bootloader 与 APP 的版本配置模块已拆开
- 文档结构已完成重构，按用途 + 主题分类
- 配置集中管理已落地（bsp_config.h / app_config.h）
- 日志服务已移植（nanoprintf + LogService）
- **ModeManager + TinyFSM 状态机已落地，模式切换功能验证通过**
- **上位机串口调试工具已添加模式切换按钮（0x10/0x11/0x12）**
- **UART2 DMA deinit 问题已修复，升级握手复位后 Bootloader 可正常启动**
- **ADC BSP/Service 第一版闭环已落地，AdcTaskA/B 可采集 512 点并输出平均值**
- **上位机串口端口下拉框刷新/选择问题已修复**
- **ADC 平均值计算异常和高频日志导致疑似卡死问题已修复/缓解**
- **调试串口关闭后进入 STM32 System Bootloader 的硬件问题已定位并通过断开自动下载控制电阻修复**
- **GSM 校准表、功率 measurement 和 TaskB 独立入口已完成第一版闭环**
- **串口助手接收区已支持跨多条复制、带时间戳复制、关键字过滤/高亮和自定义竖向胶囊滚动条**

## 近期待推进

- ADC 长时间运行稳定性继续实机验证；若仍卡死，用 ST-Link 读取 Fault 寄存器定位
- ADC 日志输出后续改为非阻塞队列或 DMA 发送，避免调试输出影响实时采集
- 业务主线（采集 -> 上传 -> 显示）待启动
- 旧项目模式迁移（POWER / MOHMS / CALI / GSM）
- 升级模块状态机优化（避免 HAL_Delay 阻塞）
