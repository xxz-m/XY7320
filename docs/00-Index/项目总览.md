# 项目总览

> [!note]
> 这页负责回答三个问题：仓库里有什么、主线是什么、现在重点在哪里。

## 仓库组成

- 下位机 Bootloader：`XY_bootloadr/`
- 下位机 App：`XY7320/`
- 上位机升级工具：`tools/XY7320Host/`
- 项目文档：`docs/`

## 三条主线

- 业务主线：采集 -> 计算 -> 上传 -> 显示
- 控制主线：上位机参数下发 -> 下位机执行 -> 返回应答
- 升级主线：APP 收包 -> 设置升级标志 -> Bootloader 升级 -> 跳转 APP

## 当前重点

历史（已完成）：

- APP 已进入 `App / BSP / Services / System` 分层
- LED 任务化拆分已落地
- 串口升级轮询已挂入任务框架
- APP / Bootloader 版本配置已各自独立维护
- 固定码握手升级链路已跑通

当前主线：

- App / Services / BSP / System / Domain / Middleware / Common 七层全部落地
- ModeManager + TinyFSM 状态机已串起 Idle / AdcTaskA / AdcTaskB 三种业务模式
- GSM 模式已闭环：AdcService → Oscilloscope → GSMPowerMeasurement → CalibrationTable
- DMR 模式第一版闭环：四通道 ADC → Oscilloscope → DMRPowerMeasurement → CalibrationTable（ADL8361 正向 + ADL8317 反向）
- 校准表统一使用 `dBm * 10` 定点格式，开关 `CALIB_ENABLE_GSM_TABLE` / `CALIB_ENABLE_DMR_TABLE` 独立控制
- 主协议升级握手 `0xF0` 已跑通；Bootloader 固定码 `XYB1/XYB2/XYB3` 保持
- 芯片命名规范化：GSM = ADL5906A；DMR 正向 = ADL8361；DMR 反向 = ADL8317
